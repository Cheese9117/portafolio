## Sobre el proyecto

Landing page de una sola página (single-page) que presenta servicios, portafolio y datos de
contacto. Construida como un único archivo `index.html` con CSS y JavaScript embebidos, sin
dependencias externas más allá de Google Fonts.

## Stack

- HTML5 semántico
- CSS3 (custom properties, Grid, Flexbox, animaciones con `prefers-reduced-motion`)
- JavaScript (ES2022+, vanilla, sin librerías)
- Google Fonts (Inter, Sora)

## Características

- Diseño responsive mobile-first
- Navegación con scroll suave y resaltado de sección activa
- Animaciones de entrada con `IntersectionObserver`
- Fondo animado en el hero con `<canvas>` (red de partículas)
- Botón flotante de WhatsApp
- Meta tags completos para SEO y redes sociales (Open Graph, Twitter Cards)
- Content Security Policy configurada

## Servicios

- Landing pages
- Chatbots de WhatsApp con IA (Twilio + n8n + Claude API)
- Automatizaciones de negocio
